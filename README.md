## 2. VerGrid

 - 대충 내가 쓰는 캘린더/시트/알람 사용하는 프로그램을 내 취향에 시트 위주로 통합한것
 - 내가 쓰는 캘린더/시트/알람을 버리고 이걸 사용하면 와 이거지 하면서 쓸 수 있게 만들기가 목표
 - "내"가 클라이언트이기 때문에 만들면서 타협할 기능이 절대 없음
